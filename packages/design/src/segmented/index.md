## API

| 参数     | 说明             | 类型    | 默认值 | 版本 |     |
| :------- | :--------------- | :------ | :----- | :--- | --- |
| ellipsis | 省略             | boolean | false  | -    |
| width    | 省略文本展示宽度 | String  | 68px   | -    |

- 更多 API 详见 antd Segmented 文档: https://ant.design/components/segmented-cn
